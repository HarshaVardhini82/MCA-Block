About the Application

Our aim is to create a system where the users can share files securely throughout the internet. We acheive this using IPFS and blockchain technology.

IPFS is a peer-to-peer protocol where each node stores a collection of hashed files. A client who wants to retrieve any of those files enjoys access to a nice abstraction layer where it simply needs to call the hash of the file it wants. IPFS then combs through the nodes and supplies the client with the file. Using blockchain ensures us that the data that is stored on the blockchain can't be deleted or tampered with. It uses protected cryptography to secure the data ledgers.

This is why IPFS is so powerful when coupled with the blockchain. Since blockchain is absolutely horrible at storing large files, we simply store the hash of the files on the blockchain where as the files itself are actually stored on the IPFS nwetwork. On top of all this we have implemented AES cryptography to to secure the hash of the file so that only intended users with the key can download the file which makes the application more secure than ever.
